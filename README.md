# BetterTransmog

**BetterTransmog** is a World of Warcraft addon that enhances the default Transmogrification frame by allowing you to dynamically resize it. Enjoy a more flexible and customizable transmog experience!

## Features

- **Resizable Transmog Frame:** Drag to adjust the size of the transmog window to your preference.
- **Improved UI:** See more items and details at once with a larger frame.
- **Lightweight:** Minimal performance impact.

## Usage

- Open the Transmogrification window at any transmog NPC.
- Drag the bottom right edge to resize as needed.
